Tailwind & CSS wiring

Create marketing/tailwind.config.ts:

import type { Config } from "tailwindcss"

const config: Config = {
  content: [
    "./app/**/*.{js,ts,jsx,tsx,mdx}",
    "./components/**/*.{js,ts,jsx,tsx,mdx}"
  ],
  theme: { extend: {} },
  plugins: []
}
export default config


Create marketing/postcss.config.js:

module.exports = {
  plugins: { tailwindcss: {}, autoprefixer: {} }
}


Ensure marketing/app/globals.css exists and contains:

@tailwind base;
@tailwind components;
@tailwind utilities;

/* optional global styles */
html,body { min-height: 100%; }


In marketing/app/layout.tsx, import the CSS at the very top:

import "./globals.css";
